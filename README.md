# WEB SITE


Um site que estou elaborando para facilitar a vida daqueles que precisam atuar com catalogo e menu.
O site possui os dados atualizados automaticamente por meio de script que é rodado via Actions, o script consiste em puxar as mudanças que acontece e cria um arquivo "JSON" totalmente atualizado que é transferido 
para o meu git pages.

Além de possuir outro script que serve para arrumar estoque, e assim que esse produto fica esgotado ele sai do site, afim de evitar clientes tentando comprar algo que não possui no estoque.
